Proyek ini untuk menampilkan portofolio penulis terutama portofolio pengolahan data untuk role data scincetist, data analytics, dan data intelligents and business.
Pengerjaan dan pengolahan data menggunakan perangkat lunak seperti Ms.Excel, Google Sheet, Tableau, dan Python.
Proses pengerjaan dan pengolahan data mencakup data cleaning, data counting, dan data visualization.
